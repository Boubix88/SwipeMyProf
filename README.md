# SwipeMyProf
This website is a Tinder-like platform based on professor profiles.

## Project initialisation
```
- ng new FrontEnd
- nest new BackEnd
```

## Launch application
- Open 2 terminals :
- ``` 
  cd BackEnd
  npm install
  npm run start:dev
  ```
- ``` 
  cd FrontEnd
  npm install
  npm run start
  ```
